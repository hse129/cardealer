# CAR MAKER 웹사이트 PRD

## 개요
사용자가 자동차 제조사(브랜드)를 선택하고, 차종, 등급, 옵션을 순차적으로 선택하여 최종 차량을 구성하고, 가격을 확인 및 저장, 여러 차량을 비교할 수 있는 웹사이트를 개발한다.

## 주요 기능
1. 자동차 제조사(브랜드) 선택 UI
2. 브랜드별 차종(모델) 선택 UI
3. 차종별 등급(트림) 선택 UI
4. 등급별 옵션 선택 UI
5. 선택한 옵션에 따른 실시간 가격 계산
6. 최종 차량 구성 저장 기능
7. 저장된 차량들 간 비교 기능
8. 반응형 UI 및 사용자 친화적 디자인

## 사용자 플로우
- 사용자는 브랜드를 선택한다.
- 브랜드에 따라 차종 목록이 표시된다.
- 차종을 선택하면 등급(트림) 목록이 표시된다.
- 등급을 선택하면 선택 가능한 옵션들이 표시된다.
- 옵션을 선택할 때마다 가격이 실시간으로 갱신된다.
- 최종적으로 구성한 차량을 저장할 수 있다.
- 저장된 여러 차량을 비교할 수 있다.
